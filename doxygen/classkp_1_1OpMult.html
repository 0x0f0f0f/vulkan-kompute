Detailed Description

template<uint32_t tX = 0, uint32_t tY = 0, uint32_t tZ = 0>
class kp::OpMult< tX, tY, tZ >

Operation that performs multiplication on two tensors and outpus on third tensor. The template parameters specify the processing GPU layout number of iterations for each x, y, z parameter. More specifically, this will be the input to ".dispatch(uint32_t tX, uint32_t tY, uint32_t, tZ)"
